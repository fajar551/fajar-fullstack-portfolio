# Fajar Habib Zaelani — Fullstack Portfolio

Repositori ini berisi situs portofolio fullstack berbasis **Next.js** (App Router) beserta aset statis dan ringkasan profil.

## Struktur

| Path | Keterangan |
|------|------------|
| `web/` | Aplikasi Next.js (UI portofolio) |
| `cv.md` | Ringkasan CV dalam Markdown |
| `foto *.png` / `*.jpeg` | Aset foto profil |

## Menjalankan proyek web lokal

```bash
cd web
npm install
npm run dev
```

Buka [http://localhost:3000](http://localhost:3000).

## Build produksi

```bash
cd web
npm run build
npm start
```
